== Description ==

With the SimpleTranslate widget you can easily allow visitors to translate your blog using this easy to widget which you can place in your sidebar.

== Installation ==

Place file in your /wp-content/plugins/ directory
and activate through the administration panel choosing what you wish to call the widget, and then go to the widget panel and
drag it to where you would like to have it!
